    Worksheet 4 Equivalence Relations and Counting

    1.

    Let \(A = \{a, b, c, d, e\}\text{.}\) Suppose \(R\) is an equivalence relation on \(A\text{.}\) Suppose \(R\) has two equivalence classes. Also \(aRd\text{,}\) \(bR c\text{,}\) and \(eRd\text{.}\) Write out \(R\) as a set.
    🔗
    Solution.
    Since \(R\) is an equivalence relation, the relation is reflexive, so every element is related to itself. Since we are told \(aRd\text{,}\) this means by symmetry that \(d R a\text{.}\) Similarly, \(cRb\) because \(bRc\) and \(dRe\) because \(eRd\text{.}\) By transitivity, we have that \(aRe\) and \(e R a\text{.}\) To make writing the elements of \(R\) easier to see, here is the equivalence relation as a diagram:
    🔗
    From this, we can more readily see that the relation will be
    \begin{align*} R = \{ \amp(a,a), (d,d), (e,e), (a,d), (d,a), (a,e), (e,a) , \\ \amp (d,e), (e,d), (b,b), (c,c), (b,c), (c,b) \}\text{.} \end{align*}

    2.

    Prove the relations below are equivalence relations, and describe the relation’s equivalence classes.
    🔗

    (a)

    The relation \(R\) on \(\Z\text{,}\) where \(xR y \) if and only if \(3x -5y \) is even.
    🔗
    Solution.
    We need to show that this relation is reflexive, symmetric, and transitive.
    🔗
    For any \(x \in \Z\text{,}\) \(3x-5x = -2x\) so this is clearly even. Hence, \(xRx\text{,}\) so \(R\) is reflexive.
    🔗
    Suppose that \(x,y \in \Z\) with \(x R y\text{,}\) so \(3x-5y\) is even. This means that \(3x-5y=2k\) for some integer \(k\text{.}\) Then,
    \begin{align*} 3y-5x \amp= 3y + 2y -2y -5x + 2x -2x\\ \amp= 5y - 2y - 3x - 2x\\ \amp= (5y-3x)-2x-2y\\ \amp= -(3x-5y)-2x-2y\\ \amp= -2k - 2x - 2y\\ \amp= 2(-k-x-y)\text{,} \end{align*}
    which is even because \(k,x,y \in \Z\text{.}\) Therefore, \(y R x\) as well. Thus, \(R\) is symmetric.
    🔗
    Now, assume that \(x,y,z \in \Z\) with \(xRy\) and \(yRz\text{.}\) Therefore, \(3x-5y\) and \(3y - 5z\) are both even, meaning that there exist integers \(a\) and \(b\) so that \(3x-5y= 2a\) and \(3y-5z=2b\text{.}\) We thus have
    \begin{align*} 3x-5z \amp= 3x -5y + 5y -5z\\ \amp= (3x-5y) + 2y + (3y-5z)\\ \amp= 2a + 2y + 2b\\ \amp= 2(a+y+b)\text{,} \end{align*}
    which is even because \(a,y,b \in \Z\text{.}\) Therefore, \(x R z\text{,}\) so \(R\) is transitive. We conclude that \(R\) is an equivalence relation.
    🔗
    To determine the equivalence classes, note that
    \begin{align*} [0] \amp= \{ x \in \Z : x R 0 \}\\ \amp= \{ x \in \Z : 3x-5\cdot 0 \text{ is even} \}\\ \amp= \{ x \in \Z : 3x \text{ is even} \}\\ \amp= \{ x \in \Z : x \text{ is even} \} \end{align*}
    where the last equality follows because \(3x\) is even if and only if \(x\) is even. (Further justification is left as an exercise for the reader :).) Similarly, note that
    \begin{equation*} [1] = \{ x \in \Z : x R 1 \} = \{ x \in \Z : 3x-5\cdot 1 \text{ is even} \} = \{ x \in \Z : x \text{ is odd} \} \end{equation*}
    where we used the fact that \(3x-5\) is even if and only if \(x\) is odd (Also left as an exercise for the reader :).) Since the sets of even and odd integers partition the set of all integers, these are the equivalence classes.
    🔗
    🔗
    🔗

    (b)

    The relation \(R\) on \(\Z\) as \(xR y \) if and only if \(x^2 + y^2\) is even.
    🔗
    Solution.
    We will show that \(R\) is reflexive, symmetric, and transitive. Note that if \(x \in \Z\text{,}\) then \(x^2 + x^2 = 2x^2\) which is clearly even, so \(x R x\text{.}\) Thus, \(R\) is reflexive. Clearly \(R\) is symmetric: if \(x,y\in\Z\) with \(xRy\text{,}\) then \(x^2+y^2\) is even. Addition is commutative so \(y^2+x^2=x^2+y^2\) is also even, meaning that \(yRx.\) Finally, let \(x,y,z \in \Z\) with \(xRy\) and \(yRz\text{.}\) Thus, \(x^2 +y^2\) and \(y^2 + z^2\) are both even, so there exist integers \(a\) and \(b\) where \(x^2+y^2 = 2a\) and \(y^2+z^2 = 2b\text{.}\) Solving for \(x^2\) and \(z^2\text{,}\) we get that \(x^2 = 2a-y^2\) and \(z^2 = 2b-y^2\text{.}\) Hence,
    \begin{equation*} x^2 + z^2 = 2a-y^2 + 2b-y^2 = 2a+2b-2y^2 = 2(a+b-y^2), \end{equation*}
    which implies that \(x^2+z^2\) is even because \(a,b,y \in \Z\text{.}\) Thus, \(x R z\text{,}\) so the relation is transitive.
    🔗
    The equivalence classes are
    \begin{equation*} [0] = \{ x \in \Z : x R 0 \} = \{ x \in \Z : x^2 \text{ is even}\} = \{ x \in \Z : x \text{ is even} \} \end{equation*}
    where we use the fact that \(x^2\) is even if and only if \(x\) is even (left as an exercise:) ), and
    \begin{equation*} [1] = \{ x \in \Z : x R 1 \} = \{ x \in \Z : x^2 + 1 \text{ is even} \} = \{ x \in \Z : x \text{ is odd} \} \end{equation*}
    where we use the fact that \(x^2+1\) is even if and only if \(x\) is odd (left as an exercise:) ). The sets of even and odd integers partition \(\Z\text{,}\) so these are the equivalence classes.
    🔗

    3.

    Describe the partition of \(\Z\) resulting from the equivalence relation \(\equiv \pmod{4}\text{.}\)
    🔗
    Solution.
    The way congruence modulo \(4\) partitions \(\Z\) is based on the remainder after division. So, there are four equivalence classes:
    \begin{align*} [0] \amp = \{ x \in \Z : x \equiv 0 \pmod 4\} = \{ \ldots,-8, -4, 0, 4, 8, 12, \ldots \},\\ [1] \amp = \{ x \in \Z : x \equiv 1 \pmod 4\} = \{ \ldots,-7, -3, 1,5, 9, 13, \ldots \},\\ [2] \amp = \{ x \in \Z : x \equiv 2 \pmod 4\} = \{ \ldots,-6,-2,2,6,10,14 \ldots \},\\ \end{align*}
    and
    \begin{align*} [3] \amp = \{ x \in \Z : x \equiv 3 \pmod 4\} = \{ \ldots,-5,-1,3,7,11,15, \ldots \}. \end{align*}

    4.

    Consider the following definitions for characters: the digits are the integers between 0 and 9 (inclusive), the letters are the lowercase letters (\(a\) through \(z\)), and the special characters are \(\{*, ?, \$, \# \}\text{.}\) Compute the number of passwords that satisfy the given constraints.
    🔗

    (a)

    Strings of length \(6\text{,}\) where characters can be special characters, digits, or letters.
    🔗
    Solution.
    The sets of special characters, digits, and letters are all mutually disjoint, which means that by the sum rule the total number of characters will be \(10 + 26 + 4 = 40\text{.}\) Then, each of the six characters in the string can be any of the \(40\) characters, so there are a total of \(40^6\) possible passwords.
    🔗
    🔗
    🔗

    (b)

    Strings of length \(7,8,\) or \(9\text{.}\) Characters can be special characters, digits, or letters.
    🔗
    Solution.
    The strings of length \(7\text{,}\) \(8\text{,}\) and \(9\) are mutually disjoint, so by the sum rule we have that there are \(40^7 + 40^8 + 40^9\) possible passwords.
    🔗
    🔗
    🔗

    (c)

    Strings of length \(7,8\text{,}\) or \(9\text{.}\) Characters can be special characters, digits, or letters. The first character cannot be a letter.
    🔗
    Solution.
    When choosing a string of length \(j\) where the first character is not a letter, there are \(10+4 = 14\) available choices for the first character between digits and special characters, and \(40\) choices remaining for the remaining \(j-1\) elements of the string. Hence, for a string of length \(j\) subject to these constraints, there are \(14 \cdot 40^{j-1}\) choices. The sets of strings of lengths \(7\text{,}\) \(8\text{,}\) and \(9\) are mutually disjoint, so by the sum rule, the total number of strings subject to the given constraints will be
    \begin{equation*} 14 \cdot 40^{7-1} + 14\cdot 40^{8-1} + 14\cdot 40^{9-1} = 14 ( 40^6 + 40^7 + 40^8). \end{equation*}
    🔗
    🔗
    🔗

    (d)

    The password must be a string of length \(6\text{.}\) Characters can be special characters, digits, or letters, with no repeated characters.
    🔗
    Solution.
    Since there are a total of \(40\) characters, and characters are not repeated, there is \(40\) choices for the first character, \(39\) choices for the second character, \(38\) choices for the third, etc. Hence, the total number of strings of length \(6\) without character repetitions is \(40 \cdot 39 \cdot 38 \cdot 37 \cdot 36 \cdot 35\text{.}\)
    🔗
    🔗
    🔗

    (e)

    The password must be a string of length \(6\text{.}\) Characters can be special characters, digits, or letters, with no repeated characters. The first character cannot be a special character.
    🔗
    Solution.
    Since the first character cannot be a special character, there are \(10 + 26 = 36\) choices for the first character. Once the first character has been chosen, then there are \(39\) choices for the second character because we do not allow for repeated characters. For the third character, there will be \(38\) choices, etc. Hence, the total number of strings of length \(6\) with no repetitions in which the first character is not a special character will be \(36 \cdot 39 \cdot 38 \cdot 37 \cdot 36 \cdot 35\text{.}\)
    🔗
    🔗

    5.

    Find a function from the set \(\{1,2,\ldots,30\}\) to \(\{1,2,\ldots,10\}\) that is a \(3\)-to-\(1\) correspondence. (The division, ceiling, or floor operations may be useful here.)
    🔗
    Solution.
    One possibility is \(f(x) = \lceil \frac{x}{3} \rceil\text{.}\) This is a \(3\)-to-\(1\) correspondence because for \(j \in \{1, 2, \ldots, 10\}\text{,}\) the numbers that will get mapped to \(j\) under \(f\) will be \(3j\text{,}\) \(3j-1\text{,}\) and \(3j-2\) (which are all elements of \(\{1,2,\cdots,30\}\)).
    🔗
    🔗
    🔗

    6.

    How many strings are there over the set \(\{a,b,c\}\) that have length \(10\) in which no two consecutive characters are the same? For example, the string “abcbcbabcb” counts, and the strings “abbbcbabcb” and “aacbcbabcb” do not count.
    🔗
    Solution.
    There are \(3\) choices for the first character in the string, and \(2\) choices for the second character. For the third, since we cannot have two consecutive characters be the same, we therefore have \(2\) choices. Similarly, there are \(2\) choices for the fourth, etc. Hence, there are \(3 \cdot 2^9\) strings of length \(10\) in which no two consecutive characters are the same.
    🔗
    🔗

    7.

    A girl scout troop with ten girl scouts and two leaders goes on a hike. When the path narrows, the troop must walk in single file with a leader at the front and a leader at the back. How many ways are there for the entire troop (including the girl scouts and the leaders) to line up?
    🔗
    Solution.
    There are \(2\) choices for the leader at the front. Then, there are \(10!\) choices made for the ten girl scouts in the middle. Finally, there is one choice for the leader at the back of the line. Hence there are \(2 \cdot 10!\) ways to line up the entire troop.
    🔗
    🔗

    8.

    Suppose a network has 40 computers of which five fail.
    🔗

    (a)

    How many possibilities are there for the five that fail?
    🔗
    Solution.
    Note that this is the same as selecting a subset of five computers from a set of \(40\) computers. There are \(\binom{40}{5}\) ways to do this.
    🔗
    🔗
    🔗

    (b)

    Suppose that three of the computers in the network have a copy of a particular file. How many sets of failures wipe out all the copies of the file? That is, how many \(5\)-subsets contain the three computers that have the file?
    🔗
    Solution.
    In this context, any such \(5\)-subset of computers that fail must contain the three computers that have the file. The two remaining computers in the subset must then be selected from the \(37\) computers that did not contain the file. Thus, the number of such \(5\)-subsets in which the three computers with the file fail is the number of ways to select two computers from the \(37\) computers without the file, which is \(\binom{37}{2}\text{.}\)
    🔗
    🔗
    🔗

    9.

    A search committee is formed to find a new software engineer to hire.
    🔗

    (a)

    If \(100\) applicants apply for the job, how many hays are there to select a subset of \(9\) applicants for a short list?
    🔗
    Solution.
    There are \(\binom{100}{9}\) ways to select \(9\) applicants out of \(100\text{.}\)
    🔗
    🔗
    🔗

    (b)

    If \(6\) of the \(9\) are selected for an interview, how many ways are there to pick the set of applicants who are interviewed? (Assume that the short list is already decided.)
    🔗
    Solution.
    There are \(\binom{9}{6}\) ways to pick the set of applicants who are interviewed.
    🔗
    🔗
    🔗

    (c)

    Based on the interview, the committee will rank the top \(3\) applicants and submit the list to the boss who will make the final decision. (Assume that the interviewees are already decided.) How many ways are there to select the list from the \(6\) applicants?
    🔗
    Solution.
    Since the applicants will be ranked, they are ordered. Therefore, there are \(P(6,3) = \frac{6!}{2!}\) ways to select an ordered list of \(3\) people without repetitions from a set of \(6\) applicants.
